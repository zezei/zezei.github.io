Latest works

  1. Present

    Freelance Full-Stack Developer

    I am currently working on a reservation system for mountains shelters of my city that allows users to browse and book available accommodation in any of.the registered mountain shelter.
    Users can view a calendar with the available dates and choose the desired number of nights to stay. The system also allows users to select the number of peoples in their group.
    The system uses MercadoPago ( which allows a variety of options such as credit card, debit card, or bank transfer) for payment processing, allowing users to securely pay for their reservation online.
    Once a reservation is made, the system sends a confirmation email to the user with details of their booking, including the dates of their stay and the services they have selected. The system also sends a notification email to the shelter administrator, providing them with the details of the reservation and the payment made.
    It provides an easy-to-use interface for both users and shelter administrators. Administrators can manage reservations, view occupancy rates, and set prices for different types of accommodation and services. They can also view payment transactions and issue refunds if necessary,etc..

    Stack Used

    Typescript Ionic Angular Firebase NestJS
  2. 2022

    Freelance Full-Stack developer

    The application is a performance analysis system for users based on their activity data from Strava.
    It utilizes Firebase Authentication for user authentication, and Cloud Functions to receive Strava webhooks. The frontend is built with Angular/Ionic. The system uses a declarative programming approach and follows agile methodology. Administrators can log in using their Gmail credentials, upload Strava segments, keep track of activity counts, obtain performance graphs and user reports, and record medical check-ups.
    Users can view their performance in specific segments and a general performance table of other users who have performed a segment. Additionally, the system stores all activities uploaded to Strava by users for later consultation.

    Stack Used

    Typescript Ionic Angular Firebase
